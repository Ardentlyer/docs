临时暴露当前目录！浏览器访问Linux服务上的文件

> 条件：未知服务器密码
>
> 作用：同一内网下，临时传文件（wget）

1、查看python的版本

```shell
python --version
```

2、构建http服务器

```shell
# python版本2.x

python -m SimpleHTTPServer 30222
```

````shell
# python版本3.x

python -m http.server 30222
````

3、在浏览器上进行访问（可以使用wget下载文件）

> ip:30222
