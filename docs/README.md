## 使用方式

- 左侧目录按多级树形结构展示，支持展开、折叠和关键词筛选。
- 文档正文会自动生成文章目录，H1-H4 标题按层级展示。
- 支持通过地址栏 `?q=关键词` 进行全文标题/路径搜索。
- 若某目录没有 `README.md`，页面会自动展示该目录下的文档索引。
